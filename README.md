# Sentiment Impact on Trading

This project analyzes how the Fear & Greed Index affects trading performance.  
It combines trade data, sentiment data, visualizations, and clustering to uncover patterns in returns, win rate, and market behavior.

## Features
- Data cleaning and preprocessing  
- Feature engineering (return %, win/loss, sentiment score, account stats)  
- Visual analysis of P&L, win rate, and trades per sentiment class  
- Random Forest feature importance  
- K-Means market regime detection + PCA visualization  

## Market Regimes
- Regime 0: Neutral / Low Volatility  
- Regime 1: Panic / Fear  
- Regime 2: Bullish / Greed Trend  

## Tech Used
Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-Learn, Jupyter Notebook.
